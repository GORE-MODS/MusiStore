# 🎧 MusiStore
The Ultimate High-Fidelity Music Streaming & Management Interface.

MusiStore is a modern, high-performance web-based music player designed for audiophiles and developers who value a tactical, glassmorphism-inspired aesthetic. Built with a focus on seamless synchronization and visual immersion, it brings a "studio-grade" feel to your local or cloud-based library.

# ⚡ Core Features
Dynamic Lyrics Sync: Real-time, karaoke-style "fill" animations that track perfectly with the vocals.

Intelligent Auto-Play: Context-aware playback that automatically keeps the vibe going with random tracks from the current album.

Tactical UI/UX: A sleek, dark-mode interface featuring frosted glass textures, responsive sidebars, and high-contrast emerald accents.

Click-to-Seek: Direct lyric-to-audio synchronization—click any line to jump the track to that exact moment.

Adaptive Backgrounds: Live radial-gradient shifts that react to track changes, creating a unique visual atmosphere for every song.

Library Hierarchy: Organized Artist > Album > Song navigation with built-in instant search filtering.

# 🛠️ Technical Stack
Frontend: HTML5, CSS3, JavaScript (ES6+)

Backend: High-performance API integration for streaming and metadata retrieval.

Styling: Custom Bezier-curve animations and background-clip text effects.

# 🚀 Getting Started
Clone the repository to your server environment.

Ensure your /api/library and /api/stream endpoints are configured.

Launch the index.html and experience your library in a whole new light.
